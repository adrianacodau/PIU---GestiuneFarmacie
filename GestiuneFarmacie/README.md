﻿# Aplicatie Gestiune Farmacie

# Proiectul consta in dezvoltarea unei aplicatii de gestiune a unei farmacii,
# realizata in C# folosind Visual Studio 2022. 
# Scopul aplicatiei este de a facilita administrarea medicamentelor 
# disponibile intr-o farmacie printr-o interfata simpla si intuitiva. 
# Utilizatorul va putea adauga medicamente noi, edita sau sterge medicamente 
# existente, precum si vizualiza lista completa de produse. De asemenea, 
# aplicatia va permite cautarea rapida a unui medicament dupa diferite 
# criterii (nume, pret sau cantitate).

# Pentru implementare, vor fi definite mai multe clase esentiale, precum 
# clasa Medicament, care va contine atributele principale (denumire, pret, 
# cantitate), si clasa Farmacie, responsabila de gestionarea listei de 
# medicamente. In plus, se poate utiliza o clasa Program pentru interactiunea 
# cu utilizatorul si controlul fluxului aplicatiei. Functionalitatile vor 
# include operatii CRUD (Create, Read, Update, Delete) asupra medicamentelor 
# si metode pentru cautare si afisare. Aplicatia va contribui la intelegerea 
# conceptelor de baza din programarea orientata pe obiecte si gestionarea 
# datelor in C#.
